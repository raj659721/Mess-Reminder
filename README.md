# Mess Reminder 🍽️

Mess Reminder is a web application designed for students to manage and track daily mess meals, reminders, and attendance efficiently.

## 🚀 Live Demo

https://mess-reminder-mess-tracker.vercel.app/

## 📌 Features

* ✅ Daily meal tracking
* ✅ Reminder system
* ✅ Attendance management
* ✅ Responsive UI
* ✅ Supabase authentication
* ✅ Real-time database integration
* ✅ Fast deployment with Vercel

## 🛠️ Tech Stack

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/raj659721/Mess-Reminder.git
```

Go to project directory:

```bash
cd Mess-Reminder
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in root directory and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## 🌐 Deployment

The project is deployed on Vercel.

To deploy your own version:

1. Fork repository
2. Import project into Vercel
3. Add environment variables
4. Deploy
